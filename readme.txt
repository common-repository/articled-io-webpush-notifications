Install and configure the Articled.io webpush notifications widget.

== Description ==
[Articled](https://articled.io/) is a leading edge service for your personal blog or website which will boost your readership and amplify your engagement by incorporating webpush notifications in a creative way.

Your readers will get access to the content they want by following your categories or publishers and automatically receiving webpush notifications to their devices.

* Automatic Process
* Unlimited Notifications
* Browser Support
* Quick Setup 

With this plugin you can:
* Sync your Articled account with Wordpress
* Install the Articled Widget
* Create a feed of your posts


== Installation ==
1. Upload `articled-io-notifications` folder to `/wp-content/plugins/` directory.
3. Activate the plugin from the Plugins menu in the Wordpress panel.
2. Add your Articled.io API Public Key and configure service worker settings.

== Frequently Asked Questions ==
= Where do I find my API Public Key? = 
At the Articled dashboard, go to account settings and copy the key from the `API Keys` section.
= I don\'t have any apps, what do I do? =
Use the Articled dashboard to setup and configure the widget, then click \'Update Apps\' in the plugin page.
